# cml-yanxuan
仿网易严选demo

包含更多端 [百度、支付宝小程序](https://github.com/chameleon-team/cml-yanxuan/tree/baidu-alipay) 分支

# 使用

- clone 当前项目
- 在项目根目录执行`npm install`
- 启动项目`cml dev`

其中地图功能 `./map/components/map` 使用了多态组件
