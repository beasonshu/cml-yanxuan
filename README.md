# cml-yanxuan
仿网易严选demo
# 安装 alpha 版本
```bash
npm i -g chameleon-tool@0.1.0-alpha.3
```
# 使用

- clone 当前项目
- 在项目根目录执行`npm install`
- 启动项目`cml dev`

其中地图功能 `./map/components/map` 使用了多态组件

> 百度开发者工具、支付宝开发者工具中需改成 “普通编译” 模式
